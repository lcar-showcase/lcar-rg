## About <a name="about">

A web-based implementation of the popular [Reversi](https://en.wikipedia.org/wiki/Reversi) board game.

#### Features:

- Singe-player mode against a computer.
- Mobile and desktop friendly.
- Designed with user experience in mind, featuring turn indicators, move history and user error protection.

## Use Instructions <a name="use">

#### Live version

Play the game at [http://lcar-showcase.github.io/lcar-rg/](http://lcar-showcase.github.io/lcar-rg/).

#### Localhost

Download or clone this repository, then run the following commands:

```bash
npm install
npm run dev
```

Once the server is running, visit the URL shown in the terminal.

#### Using the application

##### Game rules

Visit [World Othello Federation](https://www.worldothello.org/about/about-othello/othello-rules/official-rules/english) for the complete game rules.

_This implementation follows the rules of Othello - a variant of Reversi._
